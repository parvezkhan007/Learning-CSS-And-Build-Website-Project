Selector In Html/Css

Types of Selectors

1. Simple Selectors

  1. Element Selector
  2. Class Selector
  3. Id Selector

2. Universal Selector :*

3. Grouping Selector

4. Nested Selector


a. Element Selector

The element selector select HTMl elements based on the element name.

b. ID Selector

The id selector uses the id attributes of an HTML elements to select a specific element. The id of an element is unique within a page, so the id selector is used to select one unique element.

c. Class Selector

The class selectors select HTMl elements with a specific class attribute. To select elements with a specific class, write a period (.)character,followed by the class name.
